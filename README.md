### Routing information
All pages are generated by the [catch-all nuxt pattern](https://nuxt.com/docs/guide/directory-structure/pages#catch-all-route) in ```pages/[...uid].vue```. Prismic documents are fetch by matching uid document from url.
You can define custom document url according to document type in the ```nuxt.config.js``` file.

```javascript
prismic: {
    clientConfig: {
        routes: [
            {
                type: "page",
                uid: "home",
                path: "/",
            },
            {
                type: "page",
                path: "/:uid",
            },
            {
                type: "project",
                path: process.env.NUXT_PUBLIC_PROJECT_PATH + "/:uid",
            },
        ]
    },
    ...
}
```

### LOCAL SETUP
* Create a new Prismic repository [here](https://prismic.io/dashboard)
* TODO: Check if I can sync an empty [skeleton project](https://github.com/timothejoubert/nuxt-prismic-skeleton) to an existing repository. [See prismic theme](https://prismic.io/docs/technical-reference/prismic-cli#prismic-theme-options)
* Command for generate prismic boilerplate + sync with repository || Clone skeleton project [Nuxt | Prismic starter](https://github.com/timothejoubert/nuxt-prismic-skeleton)
```
npx nuxi@latest init [repository-name]
```
*  Add prismic slice machine to your project 
```
npx @slicemachine/init@latest --repository [repository-name]
```
* Update .env file from .sample.env (PRISMIC_REPOSITORY_NAME)
* Sync slice_model data on prismic repo from local slice machine
* Push on your github repo

### PREPROD ON NETLIFY
* Login in your Netlify account
* Sync a new website with your new github repo 
* Customize your domain name
* Add your environment variables in "site settings"

### WEBHOOK
* Add the generated netlify url webhook in prismic settings [Prismic tuto](https://prismic.io/docs/webhooks) | [Netlify build hook page](https://app.netlify.com/sites/hugo-tomasi/settings/deploys#build-hooks)

### PREVIEW
* Setup preview in prismic setting BO (add name, domain and path to the preview in prismic settings)
* Add the cdn link in header script and the preview path url in nuxt.config.js
```
script: [
    {
        src: `https://static.cdn.prismic.io/prismic.js?new=true&repo=${process.env.PRISMIC_REPOSITORY_NAME}`,
        defer: true,
    },
],
prismic: {
    preview: process.env.PREVIEW_PATH
}
```

### REDIRECTION
* Create ```netlify.toml``` file in static folder with specific rules [Netlify tuto](https://docs.netlify.com/routing/redirects/)
